# 📈 Stock Portfolio Tracker

A simple Python-based Stock Portfolio Tracker that helps users manage their investments by adding stocks, tracking quantities, calculating portfolio value, and viewing profit/loss.

## 🚀 Features

- Add stocks to your portfolio
- View all stocks in the portfolio
- Update stock quantity
- Remove stocks
- Calculate total portfolio value
- Simple and user-friendly interface
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- CSV/File Handling (or JSON if used)
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone this repository:

```bash

```

2. Go to the project folder:

```bash
cd Stock-Portfolio-Tracker
```

3. Run the project:

